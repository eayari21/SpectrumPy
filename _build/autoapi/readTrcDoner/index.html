readTrcDoner¶

open(fName)¶

_readS .trc binary files from LeCroy Oscilloscopes. Decoding is based on LECROY_2_3 template. [More info] (http://forums.ni.com/attachments/ni/60/4652/2/LeCroyWaveformTemplate_2_3.pdf)

fName = filename of the .trc file

a tuple (x, y, d)

x: array with sample times [s],

y: array with sample values [V],

d: dictionary with metadata
