Create elemental abundances from the user-provided list of minerals.

Args:

rocks (series): The output of :func:'object_spectra.Spectra.fetch_rocks'. This is a database of known minerals and their elemental abundances.

elems_present (float array): The relative abundance of each element in the sample, calculated in the :mod:'Spectra' initialization.

percentarray_s (float array): A sorted array of mineral abundances in the sample.

signalFloat64 Array

An initial numerical spectra free from noise.

SNRFloat64

A perscribed sigal-to-noise ratio for the added nosie

A synthetic TOF or mass spectra with background noise added throughout. This noise has the same frequency and amplitude space as the instrument electronics. This bakcground noise was derived via fourier analysis of Peridot impact spectra on the Hyperdust instrument.

A synthetic TOF or mass spectra with optional gaussian "white" noise added throughout. This white noise was derived via fourier analysis of Peridot impact spectra on the Hyperdust instrument.
